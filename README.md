## Repository Structure
이 저장소는 크게 두 가지 목적의 코드로 구성되어 있습니다.

### 1. 🔬 Research Workspace (`lab` folders)
대학원 과정 중 수행한 **비행체 탐지성 분석 및 비행 시뮬레이션** 관련 코드입니다.
* **주요 내용:** MATLAB과 Python을 활용한 탐지성 데이터 분석과 시뮬레이션
* **활용 역량:** 복잡한 공학 데이터를 구조화하고 분석하여 유의미한 결과를 도출

### 2. 📚 Self-Study Log (`lyj` folders)
엔지니어로서 전문성을 기르기 위해 학습한 **TIL(Today I Learned)** 기록입니다. 
* **주요 내용:** C/C++ 프로그래밍, 알고리즘, Kalman Filter 등
